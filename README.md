# Terron Loh Portfolio

A modern, responsive personal portfolio website showcasing my journey as a Backend Engineering Intern at DBS Bank and Cybersecurity & Digital Forensics student at Nanyang Polytechnic.

## 🚀 Live Demo

[View Portfolio](https://terronnnloh.github.io/MyPortfolio/)

## 📋 Overview

This portfolio features:
- **Hero Section** with animated typing effect
- **About** section highlighting skills and interests
- **Experience** timeline with detailed work history
- **Projects** showcase with interactive cards
- **Education** and **CCA** sections
- **Contact** form with validation
- **Responsive Design** optimized for all devices
- **Dark Theme** with cyberpunk/tech aesthetic

## 🛠️ Tech Stack

### Frontend
- **React 19** - Modern React with hooks and concurrent features
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations and transitions
- **Radix UI** - Accessible component primitives
- **Lucide React** - Beautiful icons
- **Wouter** - Lightweight routing

### Backend
- **Express.js** - Minimal web server
- **Node.js** - Runtime environment

### Development Tools
- **ESBuild** - Fast bundling
- **Prettier** - Code formatting
- **TypeScript** - Type checking
- **Vitest** - Testing framework
- **GitHub Pages** - Deployment

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **pnpm** (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/terronnnloh/MyPortfolio.git
   cd terron-portfolio
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

   The app will be available at `http://localhost:5173`
