# About My Portfolio

A modern, glassmorphic dark-themed personal portfolio website showcasing my background in cybersecurity, backend engineering experience at DBS Bank, projects, education, and co-curricular activities.

## 🎨 Features

A fully responsive, single-page portfolio application built with modern web technologies. It features:

- **Glassmorphic Dark SaaS Design** — Premium, enterprise-grade aesthetic with frosted glass cards and gradient accents
- **Smooth Animations** — Scroll-triggered fade-in effects and interactive transitions powered by Framer Motion
- **Fully Responsive** — Optimized for mobile, tablet, and desktop devices
- **Dark Mode** — Eye-friendly dark theme with cyan, blue, and purple gradient accents
- **Sticky Navigation** — Smooth scrolling navigation with active section highlighting
- **Performance Optimized** — Fast load times with Vite's instant HMR and optimized bundle

## 🛠️ Tech Stack

### Frontend
- **React 19** — UI library for building interactive components
- **TypeScript** — Type-safe JavaScript for robust development
- **Tailwind CSS 4** — Utility-first CSS framework for responsive design
- **Framer Motion** — Animation library for smooth scroll and interaction effects

### Tools & Libraries
- **Vite** — Lightning-fast build tool and dev server
- **shadcn/ui** — Accessible, pre-built React components
- **Lucide React** — Clean, minimal icon library
- **Wouter** — Lightweight client-side router
- **PostCSS** — CSS transformation and processing

### Development
- **Node.js** — JavaScript runtime
- **pnpm** — Fast, disk space-efficient package manager
- **TypeScript** — Static type checking
- **Prettier** — Code formatter for consistent style
